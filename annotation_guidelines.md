# Annotation Guidelines

These guidelines explain how dataset labels are assigned.

## Sentiment Labels

positive → expresses satisfaction or praise
negative → expresses dissatisfaction or criticism
neutral → factual or balanced statements

## Safety Labels

safe → harmless informational queries
unsafe → requests involving illegal activity, harm, or exploitation

## Topic Classification

Each text sample is assigned the most relevant topic category.

## Quality Rules

• Labels must be consistent
• Ambiguous text should be labeled neutral
• When uncertain, choose the safest interpretation
